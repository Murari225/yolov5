💡 QUICK EXAMPLES

Example 1: Web Upload
  1. Run: start_webapp.bat
  2. Open: http://localhost:5000
  3. Upload: your_image.jpg
  4. Download: detected_your_image.jpg

Example 2: Command Line
  > python detect.py --weights yolov5s.pt --source photo.jpg
  Results saved to: runs/detect/exp/

Example 3: Webcam
  > python cam_detect.py
  Press 'q' to quit

Example 4: Batch Processing
  > python detect.py --weights yolov5s.pt --source images_folder/
  Processes all images in folder

  • Model not loading → Check internet connection
  • Webcam not working → Try different source number
